# College ERP Lite

An open-source ERP platform for schools, colleges, and coaching institutes.

## Features

- Student Dashboard
- Teacher Dashboard
- Attendance Management
- Notice Board
- Role-Based Authentication

## Tech Stack

Frontend: React
Backend: Node.js + Express
Database: MongoDB


## Quick Start


### Frontend

npm install
npm run dev

### Backend

npm install
npm start
